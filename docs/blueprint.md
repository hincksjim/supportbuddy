# **App Name**: Support Buddy

## Core Features:

- Secure User Authentication: Secure user authentication via AWS Amplify v6 Authenticator. Users are identified via Cognito ID, using email, first name, last name, and postcode.
- User Dashboard: A dashboard to view summaries of past conversations and uploaded documents.
- Document Analysis and Q&A: An AI tool that answers questions about medical reports using uploaded images and PDFs, acting as a knowledgeable support companion.
- AI-Driven Conversational Support: AI-powered natural language conversation tool that answers questions about the user's condition and treatment options.
- Document Upload and Storage: Allow users to upload images and PDF documents for AI analysis and Q&A.
- Avatar Selection: Allow user to select gender and avatar for their buddy at first login

## Style Guidelines:

- Primary color: Soft teal (#7EC8E3) to inspire calmness and hope, yet avoiding the cliche of stereotypical 'teal'.
- Background color: Very light desaturated teal (#F0FAFB) to promote relaxation.
- Accent color: Muted violet (#9FA8DA) for gentle contrast, connoting trustworthiness.
- Body and headline font: 'PT Sans', a modern sans-serif font, will be used to provide a clean and accessible reading experience.
- Use soft, rounded icons with a hand-drawn feel to convey empathy and approachability.
- A clean and spacious layout with clear visual hierarchy to ensure ease of navigation and reduce cognitive load.
- Subtle, gentle animations for transitions and feedback to provide a sense of reassurance.
